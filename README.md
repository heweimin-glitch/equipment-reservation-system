# 实验设备预约系统

## 项目简介
本项目是一个基于微信小程序的实验设备预约系统，采用前后端分离架构，实现设备信息管理、预约申请、审核管理等功能。

---

## 技术栈

### 前端
- 微信小程序
- 微信开发者工具

### 后端
- Spring Boot
- MyBatis
- JDK 20
- Maven

### 数据库
- MySQL

---

## 运行环境

- 操作系统：macOS
- 开发工具：IDEA / 微信开发者工具
- JDK版本：JDK 20
- 数据库：MySQL 9.5.0

---

## 功能模块

- 用户登录与权限控制
- 设备信息查询
- 设备预约申请
- 预约审核管理
- 设备状态管理
- 个人预约记录查看

---

## 项目结构

- frontend：微信小程序前端
- backend：Spring Boot 后端
- database：数据库脚本

---

## 项目特点

- 前后端分离架构
- 基于微信生态快速访问
- 支持设备预约与审核流程
